🍯 Honey Production Analysis in the USA
📊 Project Overview

This project analyzes honey production trends in the United States using interactive dashboards built in Power BI, Tableau, and Looker Studio.

The objective of this project is to explore how honey production has changed over time, identify the top producing states, analyze market value and price trends, and understand the relationship between production efficiency and colony scale.

Through interactive visualizations, users can filter by state and year to discover patterns in honey production, stock levels, and economic value.

🎯 Project Objectives

The main goals of this analysis are:

Analyze total honey production trends in the USA.

Identify top honey-producing states.

Study supply vs demand trends using production and price data.

Explore efficiency vs colony scale relationships.

Compare dashboards built using three BI tools.

📂 Dataset

The dataset used in this project is Honey Production in the USA, available on Kaggle.

🔗 Dataset Source:
Honey Production Dataset
👉 https://www.kaggle.com/datasets/jessicali9530/honey-production

This dataset contains state-level honey production statistics in the United States from 1998 to 2012. It is commonly used for data analysis, visualization, and machine learning projects related to agriculture and production trends.

The dataset includes annual records of honey production, colony counts, stock levels, prices, and total production value across different states.

📊 Dataset Structure
Column	Description
state	US state abbreviation
numcol	Number of honey-producing colonies
yieldpercol	Honey yield per colony (lbs)
totalprod	Total honey production (lbs)
stocks	Honey stocks held by producers (lbs)
priceperlb	Average price per pound ($)
prodvalue	Total value of production ($)
year	Year of data

📅 Dataset Characteristics

Time Period: 1998 – 2012

Frequency: Annual data

Geography: United States (state level)

Number of Attributes: 8 features

Type: Time-series agricultural production datase

Insights from the map include:

Northern states produce higher quantities of honey.

Some states have smaller production due to climate conditions.

This visualization shows the economic value of honey production across states.

States generating the highest revenue from honey production

Regional differences in honey market value

4️⃣ Supply vs Demand Trends

This chart compares:

Total honey production

Average honey price

When production decreases, prices often increase, indicating supply-demand effects.

Production value

🔍 Key Insights

From the analysis:

North Dakota is the leading honey producer in the USA.

Honey production fluctuates over the years due to environmental factors.

There is a noticeable relationship between supply and honey prices.

Large numbers of colonies do not always guarantee higher yield per colony.
